# GitHub Pages Configuration

This directory contains the webDOOM project configured for GitHub Pages deployment.

## Deployment URL Structure

After enabling GitHub Pages in your repository settings, your game will be accessible at:

```
https://[your-username].github.io/[repository-name]/webDOOM/public/
```

## Quick Setup

1. Ensure `.nojekyll` file exists in the root directory (already created)
2. Push all changes to GitHub
3. Go to Settings â†’ Pages in your GitHub repository
4. Select the branch (main/master) and folder (root)
5. Save and wait for deployment

Your site should be live in 1-2 minutes!

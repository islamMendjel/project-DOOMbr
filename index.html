üéÆ How It Works

This project uses the PrBoom DOOM engine compiled to WebAssembly using Emscripten. The result is a fully functional DOOM engine that runs natively in modern web browsers.

Technology Stack:

  • PrBoom: Open-source DOOM engine
  • Emscripten: C/C++ to WebAssembly compiler
  • WebAssembly: High-performance binary format for the web
  • HTML5 Canvas: For rendering the game graphics

üöÄ Getting Started

For Players:

Once the game files are deployed, you'll be able to:

  • Play pre-loaded DOOM games directly in your browser
  • Upload custom .WAD files (Freedoom, DOOM Shareware, or licensed versions)
  • Enjoy classic DOOM gameplay with keyboard and mouse controls

For Developers:

To set up this project locally:

  1. Clone this repository
  2. Add the webDOOM game files to webDOOM/public/
  3. Run a local HTTP server: python3 -m http.server 8000
  4. Open http://localhost:8000 in your browser
